# COVID-19 Data Analysis using Python

## Project Overview

This project analyzes global COVID-19 confirmed case data using Python. The analysis includes data preprocessing, exploratory data analysis (EDA), and visualizations to understand global infection trends and identify the most affected countries.

---

## Objectives

- Analyze global COVID-19 confirmed cases
- Perform data cleaning and preprocessing
- Visualize worldwide infection trends
- Identify top affected countries
- Generate meaningful insights using EDA

---

## Dataset

**Source:** Johns Hopkins University COVID-19 Dataset

File Used:

- `time_series_covid19_confirmed_global.csv`

---

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Visualizations

- Top 10 Countries by Confirmed Cases
- Global COVID-19 Trend
- Daily Increase in Cases
- Top 20 Countries Comparison
- Histogram

- Heatmap

## Key Insights

- The United States recorded the highest confirmed COVID-19 cases.
- India was among the most affected countries.
- Global cases increased rapidly over time.
- Daily increases showed multiple infection waves.
- Confirmed cases were unevenly distributed across countries.

## Conclusion

This project demonstrates how Python and data visualization techniques can be used to analyze real-world public health datasets and derive meaningful insights.
